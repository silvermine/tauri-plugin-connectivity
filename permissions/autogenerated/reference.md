## Default Permission

Allows querying the current network connection status.

#### This default permission set includes the following:

- `allow-connection-status`

## Permission Table

<table>
<tr>
<th>Identifier</th>
<th>Description</th>
</tr>


<tr>
<td>

`connectivity:allow-connection-status`

</td>
<td>

Enables the connection_status command without any pre-configured scope.

</td>
</tr>

<tr>
<td>

`connectivity:deny-connection-status`

</td>
<td>

Denies the connection_status command without any pre-configured scope.

</td>
</tr>
</table>
